Ethereal Waves

Ethereal Waves is a music program for Linux. It uses the libcosmic toolkit for the UI and GStreamer for the backend. The source is available on GitHub.

Currently supported formats: MP3, M4A, Ogg, Opus, Flac, Wav


Planned features include:

  • Grid view
  • Condensed responsive layout
  • Drag and drop support
  • Playlist duplicate track management
  • Partial library update (only adding new tracks)
  • Exporting playlists as .m3u
  • Improved MPRIS support
  • More optional columns
  • More sorting options
  • Shuffle modes
