Ethereal Waves music player for Linux

Ethereal Waves is designed for people who want a dedicated local music player with a clean libcosmic interface, playlist tools, keyboard shortcuts, and support for common audio formats.

The current public page includes screenshots, supported file formats, installation steps, keyboard shortcuts, development roadmap items, and release history.
